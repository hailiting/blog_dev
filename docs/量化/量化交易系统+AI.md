# AI交易系统的理财顾问

## 功能

```
- backend 后端
  - api
  - core  DeltaFQ
- frontend 前端
- data 缓存
  - raw
  - results
  - simulations
  - strategies
- config 配置
```

- 回测
  - 回测历史数据展示
    - 展示信息
      - 1
        - 日期信息
        - 资金信息
        - 收益指标
        - 风险指标
        - 绩效指标
      - 2
        - 账户净值
        - 净值回测
        - 每日盈亏
        - 盈亏分布【已交易日期】
        - 调试信息【原始数据】
- 交易
  - 交易账户
    - 总资产、可用资金、持仓市值、总盈亏、总收益率、账户状态
    - 新建
      - 账户、券商、费率
  - 买入/卖出
    - 股票代码
    - 股票名称
    - 委托价格 【 买1 | 卖1 | 现价 】
    - 委托数量
  - 实时价格
  - 委托、成交、持仓
- 启动策略
  - 和交易差不多
  - 策略运行监控
    - 实时价格 | 收益率 | 今日盈亏 | 持仓 | 交易
  - 策略配置
    - 交易策略
    - 投资标地
    - 初始资金
    - 手续费率
