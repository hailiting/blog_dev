  - GDP 通胀 就业等
  - 货币政策如何影响经济增长？政府如何调控经济周期？
  - 目标
    - 生产
    - 就业
    - 物价
      - CPI 居民消费指数，PPI 生产者价格指数
- 微观经济学
  - 个体
  - 消费者如何选择商品？企业如何定价？
  - 观察对象
    - 价格、供需、消费
  - 边际效用(MU)
    - 消费者每增加或减少一个单位的商品或服务的消费，所带来的效用（满足感）的增量或减量
  - 边际成本(MC)
  - 边际收益(MR)
    - 多卖一个赚的钱，抵不上多做一个花的钱，就别做
- 计量经济学
  - 通过回归模型分析”利率变化是否显著影响投资水平？
  - 计量经济学: 将 经济理论 + 数学形式 + 统计推断 工具结合起来，通过数据对经济现象进行实证分析的社会科学。
  - 以构建模型为手段，校验经济理论、进行未来预测和政策评估
  - 数据驱动决策能力
    - 经济理论
    - 数理经济学
    - 经济统计学
    - 数理统计学
    - 例子
      - 从计量经济视角分析动量策略
        - 经济理论
          - 上涨趋势有持续性 -> 动量策略存在效力 (动量策略 -> 追涨杀跌)
        - 数学模型
          - `Profit = α + β*Signal + ε`
          - 信号条件：日跌幅>5 日交易>500万股
          - 止盈止损：收益≥10% 或 亏损 ≥ 1.8%
        - 数据处理
          - 收集价格与成交数据，生成信号，计算收益 【10年-20年】
        - 估计检验
          - 回归估计β、t-test 检验β是否显著>0
        - 回测与前瞻验证
          - 历史回测，验证稳定性
        - 结果评估
          - 查看收益、风险指标
          - 优化参数与执行流程

## 投资学与资产配置

- 传统资产：现金、股票、债券、基金
  - 现金: 活期存款、短期国债等，流动↑风险↓收益↓
  - 股票: 股票所有权的凭证，风险↑长期收益↑ -> 成长型投资者
  - 债券: 政府债、企业债、证券化债等，风险↓ 收益->
  - 基金: 股票型基金、债券型基金、混合型基金等，专业机构分散投资，风险↑ 长期收益↑
  - 非传统的替代型资产: 房地产、原材料、私募股权、加密货币、艺术品、保险等
  - 换个角度，资产类型与分析角度有关
    - 会计报表：资产账面分类
      - 流动资产 Current Assets
        - 能够在一年或一个经济周期内变现或使用的资产
          - 如现金、应收账款、存货、预付费用等
      - 非流动资产/固定资产 Non-current / Fixed Assets
        - 长期资产，很难在短期内变现或使用的资产
        - 如厂房、设备、土地、长期投资、无形资产（如专利）等
- 余额宝、零钱通 -> 被动收入
  - 货币基金: 1.15%
  - 100万的年利息为 11500
  - 日利息 = 年利息/365

### 传统

#### 债券 Bond

- 政府、企业等融资主体向投资者借钱的一种“债务凭证”
  - 本质是: 你把钱借给他们，他们定期还利息，到期还本
- 市场作用：为政府或企业提供融资渠道
  - 对投资者：是一种稳定收益、风险较低的投资方式，固收类资产
- 投资门槛：国债、地方债门槛较低，可通过银行或券商购买
  - 利率受央行政策影响，是理解宏观经济与货币政策的好窗口

#### 股票 Stock

- 股票是公司发行给投资者的所有权凭证
  - 你购买股票，就是公司股东，有权参与分红、投票表决等
- 市场作用：为公司融资提供资金支持（一级市场）
- 为投资者提供资产增值机会（二级：股价上涨和分红）
- 投资门槛：需开立证券账户（通过券商App），门槛底
  - 参与方式：打新、一级市场（上市前）、二级市场（上市后）

#### 基金 Fund

- 基金管理公司统一运作、集中投资者资金并分散投资的理财产品
  - 如股票型基金、债券型基金、混合型基金等
- 市场作用：帮助普通投资者实现专业化投资和风险分散
  - 从低风险（货币基金）到高风险（主动股票基金）的广泛选择
- 投资门槛：很低，10元起投
  - 适合缺乏专业知识但想长期理财的用户

## 非传统资产: 房地产、黄金、加密货币

### 房地产

- 通过购买房地产、商用物业、REITs(房地产信托基金)等方式
- 买入指标
  - 租售比：年租金/房价 > 3% 较为合理，衡量投资回报率
  - 房价收入比：房价/居民年均收入 = 3~6，判断居民负担能力
  - 去库存周期：现有库存/月成交量 = 6~12，反应市场热度与供需
  - 利率走势：贷款利率直接影响还款压力和房价预期，低则刺激
  - GDP/人口/基础发展周期 超市是否有增长红利
- 卖出指标
  - 房价涨幅过大、租售比过低、租金回报低于理财产品
  - 所在城市人口净流出/成交量持续下降
  - 政策信号（限购、限售、利率收紧）
  - 持有成本过高（如高物业费、税费、维护费用）
  - 自住用途变更/资金流动性需求/有更高回报率的项目

### 加密货币/数字资产

- 高风险，只投主流币
- 定投比追涨杀跌靠谱

### 保险

- 最优先配置医疗
  - 50w的重疾险（消费型、理财险）
- 先保命再宝财
  - 重疾险、意外险、寿险 > 理财型保险

## 投资组合 All Weather 全天候配置

**动态+风险评价**
多资产组合帮助稳住收益，不押注单一方向 基金 股票 大宗商品

- 对冲基金桥水基金的创始人Ray Dalio 于 1996年提出的
  - 核心：每个经济环境中选择合理的资产进行配置
- 特点：风险均衡，依据风险偏好，优化投资权重
- 收益 = 无风险收益Cash + Beta + Alpha

### Cash

- 在没有亏损的情况下，投资者可获得的回报率
  - 通常按一年期国债利率计算(1.37%)

### Beta

- 不同资产类别中超过无风险收益的超额收益
  - 股票的延期收益率为7%，则贝塔收益率为: 7%-1.37%

### Alpha

- 管理人的价值贡献部分，是偏离贝塔所获得的超额收益
  - 基金经理管理的回报为10%，大盘Beta是7%，超额为3%

### 配置策略

- 达里奥给个人投资者提供的全天候简化配置策略
  - 40% 美国长期债券
  - 30% 股票
  - 15% 美国中期债券
  - 7.5% 黄金
  - 7.5% 商品
- 2026年
  - 55% cash
  - 45% beta
  - 弱相关
