# 量化交易

- 什么是量化交易
  - 定义：用数学/统计模型和程序化系统，把“交易想法”变成可回测、可复现并自动执行的策略
  - 特性：量化强调规则化、可重复、可测量
- 流程：从构想到实现
  - 构想 Idea
  - 数据 Data
  - 特征因子 Feature
  - 模型/规则 Strategy
  - 回测 Backtest
  - 成本滑点 Cost
  - 参数优化 Parameter
  - 仿真交易 Paper
  - 实盘执行 Execute
  - 监控维护 Monitor

## 四大量化策略：基本面、技术面、统计套利、AI驱动

- 基本面
  - 原理：基于企业的财务数据、宏观经济指标、行业结构等基本面因素，建立量化因子和打分模型
    - 低市盈率（PE）+ 髙净资产收益率（ROE）
      - PE=股价➗收益 市盈率 (Price-to-Earnings Ratio)
      - ROE=净利润➗净资产 净资产收益率 (Return to Equity)
  - 因子：估值类因子、成长类因子、盈利因子、风险类因子、ESG因子(延伸趋势)
  - 优势：逻辑清晰，长期有效，和基本面研究结合紧密
  - 劣势：数据滞后、财报质量参差不齐，短期交易捕捉能力较弱
- 技术面量化策略
  - 原理：利用价格、成交量、市场情绪等市场数据，发现趋势、动量或均值回归机会
  - 因子：趋势跟随、动量策略、均值回归、成交量/资金流策略、波动率突破
  - 优势：数据实时性强，能快速反映市场情绪；对高频交易友好
  - 缺点：易过拟合，指标滞后性明显，在极端行情中失效风险大
- 统计套利
  - 原理：基于价格的统计特征和协整关系来捕捉价差机会，核心是发现价格偏离->建仓->等待回归
  - 因子：配对交易、协整检验、多资产套利、跨市场套利
  - 优势：理论基础扎实,适合中性策略（对冲市场系统风险）
  - 劣势：协整关系可能失效；交易成本、滑点对收益侵蚀大
- AI 驱动
  - 原理：利用机器/深度学习对市场进行模式识别、预测和优化，更强调自动特征提取、非线性建模、复杂交互关系
  - 因子：监督学习（随机森林）、深度学习（LSTM/Transformer）、强化学习（RL）、自然语言（NLP）、AutoML+大模型
  - 优势：捕捉非线性、复杂关系，在大数据/另类数据中表现突出
  - 劣势：黑箱性强，过拟合风险高，数据依赖严重，算力成本大

## 如何判断策略好坏：核心评估指标

- 收益类指标：累计收益、年化收益、超额收益率
  - 什么是收益类指标：投资标的在特定时间范围内的价值增长
    - 大于0则收益为正，小于0则收益为负
  - 累计收益率（Cumulative Return）时间段内的投资总增长
    - `(期末资产-期初资产)/期初资产`
  - 年化收益率
    - 将累计收益率换算成一个等效的、每年稳定复利增长的收益率
    - `年化收益率=(1+累计收益率)**(1/T)-1` `252->T = days/252 默认每年252个交易日`
  - 超额收益 (Excess Return)
    - 策略收益减去基准（Benchmark）的收益
    - ER=Rp-Rb
      - Rp = 组合收益率 Portfolio Return
      - Rb = 基准收益率 Benchmark Return
      - 假设策略赚了15%，同期大盘5%：ER = 15%-5% = 10%
- 风险类指标：最大回撤、波动率（标准差）
  - 计量投资潜在损失概率和程度的数值化指标
  - 波动率
    - 衡量投资资产价格变动幅度的指标，**波动率越大，风险越高** 波动率=√收益率的方差 年化波动率=每日波动率 x √252
    - 假设近3天收益率为1% 2% 3%, 波动率 =根号(方差([1%,2%,3%]))=1%, 年化波动率=1%x根号(252)~15.87%
  - 最大回测 Max Drawdown
    - 在一段时间内资产净值的最大跌幅
      - 最大回撤(MDD)= (最低点净值(Trough)-最高点净值(Peak))/最高点净值(Peak)
- 风险调整后收益: 夏普比率（Sharpe Ratio）、卡玛比率（Calmar Ratio）、索提诺比率(Sortino Ratio)
  - 收益风险比指标
    - 衡量单位风险所带来的回报比率，即回报与所承担风险的关系
  - 夏普比率 Sharp
    - 衡量了每单位总风险所获得的超额回报，比率越高，说明回报越高，效果越好
- 稳健性指标：信息比率（IR）、胜率、盈亏比
