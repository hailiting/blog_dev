# 股票与基金

## 在哪交易

- 线下
  - 上海证券交易所、深圳证券交易所
  - 90年代初全面实现了电子化、无纸化
  - 对于个人投资者来说，不存在线下交易的可能性
- 线上
  - 财经+交易: 东方财富、同花顺
  - 社交+交易: 雪球
  - 各大券商自营App: 中信、华泰

```shell
|投资者| -> 下达指令 -> |交易App| -> 发送委托 -> |券商服务器| -> 报送订单 -> |证券交易所| -> 自动撮合 -> |成交确认| -> 发送数据 -> |中国结算|
|投资者| <- 显示结果 <- |交易App| <- 反馈成交 <-      ^        -----------------------------|     -----------------------------|
                                                  |---------------返回结果---------------|     -----------------------------|
                                                    ^       ----------------------------清算交收-----------------------------|

```

## 如何交易

- 基本规则、流程、成本
  - 门槛
    - A股（1手）、科创/港股（50万）、美股（看平台）
  - 机制
    - A股（T+1）、港/美股（T+0）
  - 时间
    - A股：工作日上午9:30-11:30, 下午 13:00-15:00
    - 港股：早市9:30-12:00，午市13:00-16:00
  - 流程
    - 开户-银证转账-下单-成交-结算
  - 成本
    - 交易费率
      - 佣金 ≤成交金额的 0.03（万三），最低5元
      - 印花税 成交金额的 0.05%（万分之五），卖出收
      - 过户费 成交金额的 0.001%（十万分之一）
      - 经手费 成交金额 0.00341%
      - 证管费 成交额的 0.002%
    - 隐形成本
      - 买卖价差
        - 买一价和卖一价之间的天然差距
        - 流动性越差，价差越明显；流动性越好，价差越小
      - 滑点
        - 你下单时看到的价格，和最终成交的价格不一样
        - 在快速行情或大单交易时尤其明显，可使用限单价

## 和谁交易

- 市场参与者，及其博弈关系
  - 散户、机构投资者、高频交易
  - 多层博弈：
    - 横向看：不同阵营利用自身优势进行策略对抗
    - 纵向看：同一阵营内部为了有限资源进行竞争
    - 整体看：并非所有博弈都是零和博弈
  - 博弈关系
    - 零和博弈：一方收益意味着另一方损失，短线交易
    - 正和博弈：长期价值投资，从公司价值增长中获益
    - 负和博弈：高频交易、频繁交易带来的成本上浮
- 委托方式
  - 限价委托
  - 最优五档剩余撤销
  - 最优五档剩余转限
  - 对方最优价
  - 本方最优价

## 赚钱逻辑

- 系统化的选股策略与投资框架
  - 行业分析 公司分析 估值与决策 交易策略 长期主义与持续迭代
    - 宏观周期与赛道选择
    - 优质公司的定性定量筛选
    - 估值评估与买入决策
    - 仓位管理与卖出纪律
  - 行业分析
    - 如何选择正确的赛道
      - 处于有利宏观周期和产业趋势中的行业
        - 科技与高端制造
          - AI 与半导体
            - 算力基建 存储芯片、先进制程、算力中心
          - 数字经济
            - 人工智能
          - 低空经济/商业航天
          - 高端设备/工业母机
        - 新能源与绿色革命
          - 新型储能
          - 氢能与核能
        - 人口结构红利
          - 智慧养老/康复
          - 创新药与医疗器械
  - 公司分析
    - 如何选择赛道中的冠军
      - 具备持续竞争优势和良好财务状况的企业
    - 定性分析
      - 商业模式诊断、护城河评估、管理层评估
      - 公司如何赚钱？茅台品牌、华为专利等垄断性优势？
    - 定量分析
      - 财务的三张报表（资产负责/利润/现金流）、关键的综合指标
      - 负债率？毛利率？现金流净额？ROE? PEG?
        - ROE = 净利润/净资产 净资产收益率
        - PEG = 市盈率（PE）/净利润增长率（G）
          - PEG<1: 低估/合理
          - PEG>1: 可能高估
  - 估值和决策
    - 什么时候买
      - 通过估值衡量公司当前价格的合适程度
    - 估值方法
      - 相对估值: PE、PB、PEG、估值水平
      - 绝对估值: DCF（现金流折现）模型
  - 交易策略
    - 构建投资组合、执行交易计划
    - 择时？止盈止损？风险与持仓管理
  - 长期主义与持续迭代

## 基金

- 开户
  - 证券账户、基金公司直销、银行、第三方平台
- 选基
  - 投资目标（稳健、平衡、进取）、行业、基金经理
- 买入
  - 按照基金净值（T日下单，T+1确认份额）
- 赎回
  - T日申请，T+1确认赎回，T+3到账

### 交易规则

- 净值定价
  - 按“未知价”原则买入，收盘后确认价格
- T+1确认
  - 15:00 前下单，按当前净值计算，过则按第二天
- 封闭/开放期
  - 部分基金只在特定时间开放赎回
- 分红方式
  - 现金分红/红利再投，购买时勾选

#### 基金交易的手续费

- 申购费 买入时收取，通常为0.10%~1.5%
- 赎回费 卖出时收取，通常持有越久越低，甚至免收
- 管理费 按基金规模收取，日均计提，体现在净值中
- 托管费 支付给托管银行，一般 0.25%

#### 投资建议

- 避免短期、频繁地交易 -> 手续费高，容易亏损
- 长期投资 -> 申购费、赎回费被摊薄，管理费是核心影响
- 定投基金 -> 有助于分散买入时点，降低择时分险

### 基金赚钱的逻辑

- 基金经理、行业板块、盈利能力
- 理解基金赚钱的核心逻辑是“人、行业、盈利”
- 学会分析基金经理的投资能力
- 掌握行业和板块在基金表现中的作用
- 理解基金的盈利能力和长期表现的关系

#### 买基金就是买人

- 基金经理是基金投资组合的核心决策者
- 关键指标：管理经验、投资风格、管理规模
  - 年限、历史业绩曲线、价值型/成长型/平衡型

#### 行业与板块：基金业绩的发动机

- 行业是基金投资的赛道，好行业才能支撑长期赚钱
- 常见配置方式：行业基金、混合基金
  - 科技、消费、医药、新能源；跨行业配置，平衡风险

#### 盈利能力：基金能否长期跑赢市场

- 基金盈利来源：股票基金、债券基金
  - 股价上涨、分红；利息收入、债券价格变化
- 评价指标：夏普比率、最大回撤、波动率
