## 什么是 TypeScript

TypeScript 是一种由微软开发的自由和开源的编程语言。它是 JavaScript 的一个超集，而且本质上这个语言添加了可选的静态类型和基于类的面向对象编程。

## 类 C 语言特性

- 1. 强类型的编程语言，显示声明字符串
- 2. 常量、变量、作用域、this、可空类型、真实数组、结构、枚举
- 3. 面向对象 类、继承、多态、接口、命名空间、变量的修饰、构造函数、服务器（get, set）、静态属性
- 4. 委托、泛型，反射，集合(多态数组)，匿名方法， 拆箱
- 5. 多线程 webwork
