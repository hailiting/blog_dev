# 判断字符串是否包含某个字符串
## String对象 方法
###  一：indexOf()
~~~
"12345".indexOf("3")> -1? true: false;
~~~
### 二：search()
~~~
"123".search("3")>-1? true : false;
~~~
###  三：march()
~~~
"123".match(/3/) ? true: false;
~~~
## RegExp对象方法
###  四：test()
~~~
RegExp(/3/).test("123") ? true: false;
~~~
### 五：exec()
~~~
RegExp(/3/).exec("123123") ? true :  false;
~~~
