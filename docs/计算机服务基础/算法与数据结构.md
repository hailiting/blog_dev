## 链表结构

### 链表是什么

- 链表是一种物理存储单元上非连续、非顺序的存储结构
- 链表的逻辑顺序是通过链表中的指针链接次序实现的

```java
// 单向链表
class ListNode {
  int val;
  ListNode next;
}

// 双向链表
class ListNode {
  int val;
  ListNode next;
  ListNode prev;
}
/// 环形链表
// 首尾相连
// 局部成环
```
