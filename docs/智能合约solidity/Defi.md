# Defi

- 复利率计算

```js
1000 --一年后--> 1100
1000 * 1.1 * 1.1 * 1.1
1000 * 1.1^3
```

- FV = Future Value 终值
- PV = Present Value 现值
- r = annual interest rate 利率
- n = number of periods 周期
- t = years time 年

* $FV = PV * (1+r)^n$
* $FV = PV * (1+r)^{1*5}$
* $FV = PV * (1+\frac{r}{4})^{4*5}$ 一年复利 4 次
* $FV = PV * (1+\frac{r}{n})^{n*t}$

* 复利 Compound Interest
