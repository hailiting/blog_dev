# 项目导航

## node 版本

v12.22.1

## 启动流程

启动 xampp 8080 端口

- npm run client:deves5
- npm run server:dev
- npm run server:start

http://localhost:8082/books/list

## 亮点

- mpa 添加 spa 切换
- 使用 DI(依赖注入)实现 ioc(控制反转)思想
- bigpipe 处理大的页面数据
- 仿 fundebug 错误处理，服务端，客户端
- 安装 postcss-preset-env 使用 nextcss
- service 增加服务， 采用依赖注入方式管理 node 代码
