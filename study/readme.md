# 区块链学习路线图

## 第一阶段：基础准备（必备基础）

1. 区块链开发环境搭建
2. go 语言基础实战
3. 3 小时熟悉 Docker
4. Go 语言设计模式与追妹子

## 第二阶段：区块链入门（理解原理）

5. Python 实战区块链与共识算法
6. Golang 七步实现模拟比特币

## 第三阶段：以太坊学习（智能合约开发）

7. 区块链 2.0 以太坊智能合约与 DApp
8. 以太坊钱包开发-交易所开发基础
9. 智能合约首创区块链 2.0 以太坊源码分析
10. 实战以太坊智能合约虚拟机开发

## 第四阶段：比特币深入

11. 实战比特币国密修改与比特币钱包开发-交易所基础
12. 分析比特币源码定制山寨币

## 第五阶段：联盟链

13. 联盟链之王-超级账本入门
14. 联盟链之王超级账本智能合约与 DApp

## 第六阶段：高级定制（源码级别）

15. 以太坊源码分析与国密定制
16. 以太坊源码定制国密与修改共识算法为 dpos

## 第七阶段：部署运维

17. docker 部署实战
18. docker 与 k8s 基于 Golang 深度实战

## 第八阶段：扩展技能

19. 数据挖掘入门视频课程

---

## 学习建议

- 前 4 个课程是基础，务必扎实掌握
- 阶段三和阶段四可以根据兴趣选择先学以太坊还是比特币
- 源码分析类课程难度较高，需要前面的基础打牢
- 建议每学完一个阶段做一个小项目巩固
