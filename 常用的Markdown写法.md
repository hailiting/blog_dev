# 常用的Markdown写法
## 打出上标和下标
$x^p_{ij}$
~~~
$x^p_{ij}$
~~~
1. 公式的编辑都在两个符号中间
2. ``^``符号后接的字符为上标
3. ``_``符号后接的字符为下标
4. 如果同时有两个下标，则需要使用``{}``来将符号括起来
